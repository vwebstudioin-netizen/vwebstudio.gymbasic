# 🏋️ Gym & Fitness Studio — Basic

A sleek, bold, and **fully fluid** gym & fitness website built with **Next.js 14**, **Tailwind CSS**, **Framer Motion**, and **Firebase**. Deployed on **Vercel** for $0/month.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ and npm

### Installation

```bash
# Clone the repo
git clone <your-repo-url>
cd gym-basic

# Install dependencies
npm install

# Copy environment variables
cp .env.local.example .env.local
# Edit .env.local with your Firebase credentials

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## 📄 Pages

| Page | Route | Description |
|------|-------|-------------|
| Home | `/` | Hero, intro, featured classes, trainer spotlight, CTA |
| Programs | `/programs` | Filterable class cards with detail modals |
| About | `/about` | Gym story, stats, values, facility gallery |
| Contact | `/contact` | Contact form, info, map, hours |

## 🎨 Design

- **Colors:** Electric Red `#e63946` + Neon Orange `#ff6b35` on Midnight Black `#0d0d0d`
- **Fonts:** Oswald (headings) + Inter (body)
- **Animations:** Framer Motion scroll reveals, hero zoom, spring physics

## 🔧 Tech Stack

- Next.js 14 (App Router)
- TypeScript 5.5
- Tailwind CSS 3.4
- Framer Motion 11
- Lucide React Icons
- Firebase Firestore (contact form)
- Vercel Hosting

## 📦 Deployment

Push to GitHub → Connect to Vercel → Auto-deploy.
